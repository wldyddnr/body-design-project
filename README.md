## 프로젝트 기능 구현

__1. 로그인__
  - 아이디,비밀번호 찾기 - 이메일로 아이디,비밀번호 전송
   
__2. 회원가입__
  - 유효성 검사, 아이디 중복체크
  - 이메일 인증
   
__3. 마이페이지__
  - 회원이 가입할때 입력한 정보가 출력되고 현재 비밀번호를 입력 후 정보를 변경 가능
   
__4. 칼로리 계산기__
  - 나이,키,몸무게,성별,활동량 입력후 대사량 계산,저장
  - 저장한 정보 출력
